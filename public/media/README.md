# Media replacement guide

Replace the fallback art direction with your own licensed assets for launch.

Recommended files:

- `hero/hero-desert.mp4` — widescreen cinematic video, 12–20 seconds, muted, loopable
- `hero/hero-poster.jpg` — lightweight poster image for the video
- `camp/luxury-camp-01.jpg`
- `camp/luxury-camp-02.jpg`
- `camel/camel-sunset-01.jpg`
- `camel/camel-sunrise-01.jpg`
- `quad/quad-ride-01.jpg`
- `tours/4x4-private-01.jpg`
- `gallery/gallery-01.jpg` to `gallery/gallery-12.jpg`

Recommended image export:
- JPG or WebP
- 1800–2400px wide
- compressed for web
- warm grading for visual consistency
